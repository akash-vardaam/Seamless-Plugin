Elementor widgets and dynamic tags for Seamless events and memberships.

== Description ==

Seamless Addon extends the Seamless plugin with powerful Elementor integration:

* **Event List Widget** - Display events with list/timeline, grid, or calendar layouts
* **Membership List Widget** - Display memberships with beautiful card layouts
* **Dynamic Tags** - Insert event and membership data dynamically into any Elementor element
* **Single Page Support** - Automatically load event/membership data based on URL slugs
* **HMVC Architecture** - Clean, maintainable code following WordPress best practices

= Requirements =

* WordPress 6.0 or higher
* PHP 7.4 or higher
* Seamless plugin (active)
* Elementor plugin (active)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/seamless-addon/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure Seamless and Elementor plugins are active
4. Configure settings under Settings > Seamless Addon

== Frequently Asked Questions ==

= Does this work without Elementor? =

No, this plugin requires Elementor to be installed and active.

= Does this work without the Seamless plugin? =

No, this plugin extends the Seamless plugin and requires it to be active.
